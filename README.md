# git-88433

## Comandos básicos

Adicionar arquivo para commit.
```
git add .
```

Realizar "commit" (versionamento)
```
git commit -m "Mensagem"
```

Atualiza repositório local com dados do repositório remoto.
```
git pull
```

Envia dados para repositório remoto (repositório no GitHub)
```
git push
```
